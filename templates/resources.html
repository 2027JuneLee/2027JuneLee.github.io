About Us!

MathDoneMad is a special project created by June (Junsoo) created within a 2-week time limit. It aims to give users an AI learning guide (fully integrated ChatGPT), curated resources, and a simple UI to help students survive AP Calculus and the SAT's math section! This is intended to be used to learn concepts, catch weak areas, and get unlimited practice questions for free and with ease. Unlike other websites, this is completely free and has minimal interruptions.
